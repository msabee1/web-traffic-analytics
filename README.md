# 📊 Web Traffic Analytics Project

## 📌 Overview

This project analyzes website traffic data to understand user behavior, engagement, and drop-off patterns.

## 🎯 Objective

To evaluate digital product performance and improve user experience and conversions.

## 🛠️ Tools Used

* Python (Pandas, Matplotlib)
* Jupyter Notebook

## 📊 Key Analysis

* Traffic Source Analysis
* User Engagement Analysis
* Bounce Rate Calculation
* Funnel Analysis (Sessions → Engaged Sessions)
* Time-based Analysis

## 🔍 Key Insights

* High variation in engagement across traffic sources
* Significant drop-off between sessions and engaged sessions
* Peak user activity observed during specific hours
* High bounce rates indicate UX improvement areas

## 🚀 Business Recommendations

* Optimize low-performing traffic channels
* Improve landing page experience
* Focus on high-engagement sources
* Align campaigns with peak user activity

## 📂 Files Included

* Notebook (.ipynb)
* Dataset (.csv)
* Insights Report (.pdf)
